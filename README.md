# Neuromelanin
Collection of neuromelanin analysis pipelines

## NM_toolbox
[NM_toolbox](NM_toolbox/) is a (private) MATLAB based pipeline using MNI warped NM with hist-mode CC mask reference for CNR.

Fork of Columbia University Tech Ventures (Cassidy, Horga, Wengler) on https://github.com/LabNeuroCogDevel/NM_toolbox
